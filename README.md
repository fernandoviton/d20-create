# d20-create

An app for creating and managing d20 characters

## Installation

```bash
npm install electron -g
git clone https://github.com/fernandoviton/d20-create.git
cd d20-create
npm install
npm start
```

### Tests
`npm test` or for continuous `npm test -- --watch`
